关于压制后的录像在索尼相册中排序错乱问题的治疗方案

发表于 2017-02-18 | 分类于 3C & 数码 | | 阅读次数:
字数统计: 239

出去玩，拍照，录像，回到家感觉录像占的空间太大（比特率高达18M）于是压一下，码率压到4M，再丢回手机之后，录像就在最前面了（按时间顺序排列的

特别是，把几个月之前的录像压了再放回去，时间错乱得更多，不能忍

试着调整创建时间、修改时间来治疗此问题，无效

最后发现，是mp4的元数据里面的编码时间决定了此视频在相册中的位置

搜了一下，果然有蛋痛之人遇到过这种问题：https://superuser.com/questions/523286/how-to-make-handbrake-preserve-capture-time-creation-time

解决方法是用FFmpeg来把原始视频文件的元数据完全的复制到压缩后的视频文件中去，真机智！

命令行是：ffmpeg -i 原文件.mp4 -i 压缩后的文件.mp4 -map 1 -map_metadata 0 -c copy正确时间的文件.mp4

望江大公仔使用感想及记录，22750公里更新

发表于 2016-11-12 | 分类于 机车 & 旅行 | | 阅读次数:
字数统计: 2.3k

到手两周跑了1300度公里，说几点初步使用感想

1.低扭很给力，烂路爬坡，转速2000不用半离合，发动机憋不死

2.相对应的高转无力。。。和华洋160的宗申发动机相比，给油之后，把手传来的感觉，大公仔像是小姑凉在拉你的手，华洋160像是黑叔叔一把把你拽过去。。。。（毕竟华洋是7000多块的车，而且没有任何直流电系统。。。

3.大灯是渣渣，自己改了LED大灯和LED射灯之后好多了

4.油耗太良心了，油箱5.1L（最多5.2L，并非使用手册上说的5.5L）能跑210公里（磨合期，保守方式驾驶，如果 激进驾驶+爬坡多，190公里）

5.油箱略小。。。而且没有副油箱，跑长途心慌啊。。。

6.油表是扯的，140公里就开始闪低油量报警，一闪就闪70多公里

7.大灯总成插头松过一次，我还以为是自己接线没接好呢，结果是插头的母卡太松了，导致公棒插入后卡不紧接触不良。。。。。希望点火器卡子不要也想这样抽风。。。

8.前后减震都略硬，跑公路是完全没有问题，不过乡村公路就有点吃力了，基坑道和稍微大点的不规则碎石路简直噩梦，完全跑不起速度

9.速度表快了，实际速度为表显速度*0.91 【后有修订】

10.目前没出磨合期，发动机震动的话，感觉速度50非常良好，速度60稍微有点大但是还可以接受，但是后视镜都没震得发花，更高的速度就不知道了没有试过

11.没有脚启动，内心有点不安，如果哪天遇到打不燃，把电瓶电打空了，就只能推车了。。。。

12.反光镜设置不合理，有一半都是自己的胳膊。。。

13.空挡前300公里基本上是找不到的，300之后改善很多，1000之后非常容易找了（也有人为因素，熟练了）

14.没有双闪，转向灯没有提示音，没有防盗接口，没有大架

15.后刹车刹车的时候会发出尖锐的鸡儿鸡儿声。。。。

16.车头仪表板的时钟每天会慢一两分钟

总体相对于价格来说，非常满意，存在的问题都不是大问题而且可以接受，代步、近郊遛弯非常适合

急性高山病相关药物以及饮食注意事项

发表于 2016-02-08 | 分类于 运动 & 户外 | | 阅读次数:
字数统计: 531
1.乙酰唑胺（已停产）新药为 醋甲唑胺
加速碳酸盐排泄，提高血液酸性，帮助呼吸，加速适应过程，每天2次，每次125mg，进山前1天开始服用，进山后两天停用，对肺水肿、脑水肿无效
2.硝苯地平

钙离子阻断剂，缓解肺动脉收缩，每12小时20-30mg，缓释为宜，预防和缓解肺水肿，起效较慢，导致低血压，不建议登山使用

西地那非，缓解肺水肿，俗称伟哥，贵。。。

吸入沙美特罗（治疗哮喘药物），预防肺水肿，贵。。。

3.地塞米松（部分人群对此过敏）

类固醇药物降低颅内压，每6小时4mg，能在给药后6小时内起效，缓解脑水肿，赢得下撤时间

4.阿司匹林

进山前1天，每天3次，每次320mg，直到出山，缓解头疼

或者布洛芬（非处方药

5.维生素C

每餐后250mg，维生素E，每餐后100UI，缓解急性高山反应症状

6.水

补充脱水，维持血浆总量，维持携氧能力

7.少吃含有亚硝酸盐和亚硫酸盐的食物

避免偏头疼，例如 奶酪、巧克力、柑橘、卤肉、味精.

8.白天食用易消化、高GI值食物（单糖、碳水化合物）

少食多餐，减少肠胃负担供血需求

9.晚上食用蛋白质、脂肪含量高的食物

以持续供能度过漫漫长夜

如果食物单一注意补充维生素、电解质（每升水添加1.5g低钠盐，可同时补充Mg、K）

备注：

1-4均为处方药，但是管控不严格，在网上药房可以买到

2,3对脑水肿和肺水肿只能缓解，并不能治疗，需及时下撤
