Lychee--荔枝图床-docker-部署-记录

Posted on 2018-12-05 | In 3C & 数码 | | Visitors:

简介

Lychee
是一款开源的图片管理系统，实际上也可以用作图床，一直在更新，自身功能很强大，还支持二次开发插件及扩展来丰富功能，此外这套系统也支持
Docker 。

官网地址

项目 Github 地址

Demo 演示

Docker

部署

以ubuntu16.4为例

安装docker环境

sudo apt-get install docker.io

拉下镜像

sudo docker pull kdelfour/supervisor-docker

建立文件夹并设置权限（可跳过）

cd /root

mkdir lychee

cd lychee

mkdir data

mkdir uploads

mkdir mysql

mkdir uploads/import

mkdir uploads/big

mkdir uploads/thumb

chmod -R 777 uploads/ data/ mysql/

如果不手动建立，就不要给容器指定外部目录，否则会报错权限不足[1][2]

Unable to save this configuration. Permission denied in \’data/\’.
Please set the read, write and execute rights for others in
\’data/\’and \’uploads/\’. Take a look at the readme for more
information.

如果有其他报错，可以进入lychee后点击左上方小齿轮，点击Diagnostics查看错误

报错后，修改文件夹权限之后，可使用如下命令重启容器

查看正在运行的容器[3]：

docker ps

查看所有容器[3]：

docker ps -a

重启容器[3]：

docker start \

运行镜像

sudo docker run -it -d -p 80:80 -v /your-path/uploads/:/uploads/
-v /your-path/data/:/data/ -v /your-path/mysql/:/mysql/
kdelfour/lychee-docker

替换your-path为刚刚新建目录的地址，例如在/root下的话就是：

sudo docker run -it -d -p 80:80 -v /root/lychee/uploads/:/uploads/ -v
/root/lychee/data/:/data/ -v /root/lychee/mysql/:/mysql/
kdelfour/lychee-docker

如果跳过了刚刚的建立文件夹及权限设置，数据库及图片将会保存在容器内部，则使用下列命令运行镜像：

sudo docker run -it -d kdelfour/lychee-docker

设置自启动

在运行docker容器时可以加如下参数来保证每次docker服务重启后容器也自动重启[4]：

docker run

--restart=always

如果已经启动了则可以使用如下命令[4]：

docker update --restart=always \

配置

运行镜像后，浏览器中输入服务器IP以登录

首次登录需要初始化数据库，使用如下配置

user name : lychee

user password : lychee

然后就是设置管理账号与密码，设置一个足够复杂的密码就可以了

其他事项

设置图片公开，可以得到一个直接可访问的链接

然而，对此照片取消公开，并不会使此链接失效……

意味着，只要设为公开，发布了此链接，就永远公开了，想取消公开只能删除图片

不过，可以使用WinSCP、Xftp、等其他方式，替换掉
/your-path/lychee/uploads/big/
下面的对应文件，在不更换图片链接的情况下，对外链的图片进行修改

挡风小了，换了个435mm高的加大挡风，好些了不过还是有风从挡风的两侧绕过来的湍流吹到头

车把太靠前了，可能是我手短，装了一个加高后移的座子，还是感觉不够。。。

要蛋蛋贴着油箱坐，手臂才是一个比较舒服的弯曲姿势

推荐使用这种快速接线端子，改开关分组方便，年审拆灯方便

达到的效果是：

细话：

美亚、史带这两个系列的户外保险都是使用的旅行意外保险的条款（美亚旅行意外伤害保险基本条款，旅行人身意外伤害保险条款），然后采用一些批单来扩展该条款的使其具有户外运动的特性，但是承保内容主题依然是旅行中发生的意外，而不是户外运动中的意外。

这两份条款中，关于旅行开始的界定都是这么描述的：

离开其境内日常居住地或日常工作地所在的市级行政区域或搭乘公共交通工具直接前往其境内日常居住地或日常工作地所在的市级行政区域之外的旅行目的地

同时，美亚的驴行天下、史带的那3个都出了一个批单：美亚扩展承保个人短途旅行批单，史带扩展承保市内批单，批单中把上述描述中的 市级行政区域 去掉了，意为扩大承保范围以涵盖短途的室内的旅行。

但是，我认为这导致了本身明确的界限的模糊化，如何界定 日常居住地 呢，我出了我家门算不算离开日常居住地，还是要出我家小区？还是出我们这个街道社区？

另外，适用于美亚畅游神州的 美亚旅行意外伤害保险仅承保境内旅行批单 中没有把
市级行政区 去掉，所以美亚畅游神州还是得离开日常居住地的城市。

在电话咨询美亚、史带之后，他们都做出了明确表示，重点在上文中的 旅行 而不是离开日常居住地；只保旅行中的意外，而以锻炼为目的日常的户外运动是不在承保范围之内的。

美亚在电话沟通中提出了旅行的定义，以游览观光娱乐为目的的外出就是旅行。同时客服表示，每天以锻炼身体的跑步，是不保的；每天以游览观光娱乐为目的的爬山，是承保的。

Mavic Pro-Air-大疆无人机使用心得体会

Posted on 2018-04-10 | In 3C & 数码 | | Visitors:

1.记得插tf卡，插电脑上考出来之后马上插回去，不然迟早得忘

2.起飞、降落的时候拿着屁股对着自己，不然的话脑子一抽心一慌打杆反了妥妥的炸

3.非恶劣天气下，高空飞行建议取下云台保护罩，减轻飞行重量，避免画质损失。关键是从高空炸鸡炸下来那货也保护不到什么。。。。（恶劣天气、低空作死飞行可以带上

4.时刻关注电量，记不住的话把低电量报警调高点，等开始报警了嫌吵的话再调低点都可以；电量条上的最低返航电量是有一些富裕的，但是为了安全起见，避免遇到意外情况时却没有足够的电量了，最好还是听从其建议。

5.关注电池电压（可在设置中打开显示到屏幕上），大幅度打杆会导致大电流放电，电压会掉很多，希望电池寿命长点、飞行续航长点的话应该避免，比如S模式全速向前全速爬升一会儿就电池过流警报了，自己看着电压打杆就不会触发放电过流警报

6.起飞前观察环境，风向，阳光方向，自己所站的朝向

风向会影响返航时的耗时，如果顺风出去，回来的时候就会耗时更久。强烈的阳光会触发前视觉避障，如果太阳很大，最好是把自动返航避障爬升关了（前提是做好飞行计划，见后一条）。自己所站的朝向、以及飞机所在的位置，要结合实际地形、指南针、地图来综合考虑，根据飞机的位置调整自己（也是遥控器天线）的朝向，以获得最好的信号。

7.起飞前做好飞行计划，想好自己往哪个方向飞，路径上有哪些阻隔，返航高度应该设为多少。

起飞之前估计设置一个返航高度，飞出去之后，尽快爬升到所有障碍物之上，然后记下高度，再根据实际需要重新设定返航高度，设高了浪费电划不来，设低了有危险。

8.天线位置是在前桨臂上，如果希望获得更远的遥控和图传范围，可以选择爬高后倒飞或侧飞，不过需要千万确保没有障碍物

9.如在计划之外，临时起意想拍摄视频、照片，一定记得确保确定自己的航线上没有障碍物，特别是向下、向后、侧向移动时的拍摄，建议先原地360转一圈看看，再向下45度转一圈看看，再向下90度看一眼。

如需要下降的时候，可以调转机头朝自己所站的位置看一眼，看下降多少会有阻隔，尽量保持在安全高度之上，虽然失控了会自动爬升，但是会干扰飞行和拍摄。

10.下降速度只有3米/秒，如果飞高了，返航的时候（在确保安全的前提下），记得全程下降打满，尽量避免已经飞到自己头顶了，却还有400米的高度需要下降。。。

11.建议自己做一个
起飞前检查清单，每次起飞的时候照着清单走一遍，费不了多少时间，但是很有帮助

12.电池充电时温度会升高，如果升高到一定程度会自动减少电流以避免损伤，所以可以用一个小风扇吹着电池给他充电。

13.电池的参数写的最大充电功率100W，而官方标配的220V充电器输出功率只有50W，应该是从慢充升温小保护电池的考虑上出发的，可以配合三方大功率充电器和小风扇一起使用

总体费用概览

车辆购置税

交强险120元，车船税9元

路桥67元（10月25日交的，两个月多一点）

临时牌照5元

车辆号牌费、行驶证工本费、登记证书工本费 55元

120+9 + 67 + 5+55 = 256

前文

我是在渝中区办的，其他地方流程相同，自行寻找地址

文中的车辆资料复印件是指
车辆合格证、一致性证书，因为我都复印了很多份带着的，他要啥我就给啥，所以也记不得每个步骤中他到底是要的
车辆合格证 还是 一致性证书 还是 都需要。。。反正多复印几份就是了

取得发票

建议让商家按官方报价的6~7.5折开发票，强行坚持此要求

如果不行，也是能开多低开多低

交购置税

重庆市国家税务局车辆购置税征收管理分局
重庆市九龙坡区渝州路86号，石油路彩电中心旁边

需要身份证复印件，购车发票第三联，车辆资料复印件一份

里面有个电脑，自己坐上去挨个填空，然后打印出一张报税申请表，或者在外面让打印店打印也可以

购置税的算法是，发票价格剔除增值税之后的不含税车价，乘以10%

具体算法是，比如发票是12000元，购置税就是12000/1.17*0.1=1025.641元

而不是代办上户人口中的12000*0.1=1200元，他们一般都会吃这个差价

另外，在计税系统中，有个计税最低价格，如果不含税车价（发票价格剔除17%增值税之后的价格）低于计税最低价格，就以计税最低价格计算购置税

到我这情况，官方报价32000的车，发票开的是27200，算出来不含税价格是23247，报税也就是按23247报，所以交税2324

最后报税的时候，工作人员给我说，我这车的最低计税价格是20500，最低购置税就是2050，相当于我因为发票的原因多交了274元

保险

身份证复印件，车辆资料复印件

一年120元，代收9元的车船税

不需要拓号。。。我还辛辛苦苦的拓了号，结果说不要，就给扔了，但是第二天上户的时候又要拓号。。。。。。

因为保险是第二天生效，所以等到第二天才能去办上户

上户

我是在渝中区车管所上户的，据说此车管所人最多最挤，但是验车也相对比较松，如果新车没改装啥的话，建议去其他车管所上户，改装之后年审再去渝中区车管所

1.交材料

需要提交的材料有
车辆登记申请书、身份证复印件、车辆合格证原件、购置税完税证明、交强险保单第三联

首先在1号窗口交材料，他们会把这些资料录入系统电子化，然后在系统里找一个你车型的标准照片打出来给你附在材料后面

最左边靠窗就是一号窗口了

2.验车拍照

然后拿着材料去负一楼，不用在外面排队，直接下去，找个空位插队跟着前面的人走

如果装了边箱尾箱什么的，多半都验不过，最好拆了再去

我装了个保险杠，还有几张贴纸，倒是验过了

他们会把你材料收走，然后你就可以回楼上了

3.选号、交费

上楼去2号窗口刷一下身份证，等着底下验车的把资料送上来

看着大厅的电视，出现自己名字后，去4号窗口拿选号的单子

他给你单子后，去门口的自助选号机器选号

只有180秒时间，而且全部是渝D，号码是5位纯数字，大多数都带4，也看运气了，另外个朋友选到XYZZZ不带4的号码

选号后到负一层，4号窗口查今日应缴纳的路桥费用，然后1号窗口交路桥费（只收现金），再回4号窗口领路桥卡，最后到2号窗口交上户费（只刷卡）（真他妈折腾。。。

回到楼上4号窗口，让他们打一张临时牌照出来

然后就可以走了，隔几天去领正式牌照

关于结组

我觉得登山手册里面说得很对，表达能力有限，直接上图。。。

年保结组不结组，带不带保护器材，我觉得还是取决于 季节、人员、攀登的线路

至少在9月中秋节前后，雪层厚度和硬度是非常适宜的，全队人员体能、技能相当，如果避开裂缝威胁选择合适的冲顶线路，坡度都不陡，不容易滑坠，而且即使滑坠了，也可以很安全迅速的制动，所以完全没有必要使用结组和保护器材，只会徒增负重，降低进山的行进速度、冲顶的速度，以及丧失对天气变化处理的灵活性。

举个例子，如果使用结组、保护，登顶需要4小时，下撤需要3小时，一共就是7小时，而在7小时内，天气很有可能变坏（不管上午天气如何，基本每天下午都会下点雨/雪），而要保证下午不被雨雪困扰，那就得早上早早的出发，而出发早了气温又低，云雾又没散开，增加行进难度。

而轻装不结组不设保护点冲顶，完全可以等上午10点太阳出来得大大的，一上一下下来吃午饭刚刚好。。。

总之，结组不结组，带不带保护器材，应该是一个需要事先思考的问题。

9月13日，晴，多云，阴，雪，雪雾

起床就是晴天，真棒！

这么好的天气，赶紧再爬一遍。。。

山谷中稳定的雾气，不抬升，就不会变天

但是高层的卷云预示这未来几天天气不妙

掏出锉刀赶紧锉冰爪，哪知道一根筋上来了，还有就是工匠情怀上来了，一口气锉了一个小时锉得比出厂还锋利。。。。。。。（真蛋疼！

导致9点半才出发。。。

上升路线基本和昨天下撤路线一致，沿着区域3和4之间的无裂缝区域上到区域5右侧，完全绕过区域5不去过雪桥

这是区域3的冰裂缝。。。我都不好意思叫他们冰裂缝了。。。

这是区域4左侧的冰裂缝，高度落差不大，但是有点宽，这雪桥还是不爬为妙

走到区域5右边的时候，感觉那边可以爬的样子，完全不用过裂缝，只是滑坠了后果有点严重，直接掉缝里去了。。。。

后略不表，无言说价值

下撤后才中午12点半，回到营地煮了碗汤，吃了块压缩饼干之后昏昏欲睡

不行，还有这么一大下午呢，于是又收拾东西去旁边那个冰川发育主方向的坡去遛弯

打算联系下各个方向的滑坠制动，哪知道雪不够硬，深度又合适，坡度又不够，就是不穿冰爪，怎么都摔不倒。。。

故意摔倒了也滑不动。。。还得努力向下摔倒。。。。。

好蛋疼。。。。

于是又回到营地继续烧水。。。。

9月14日 雪雾，多云，长时间阵雨

打算今天下撤了，脸晒得痛了，压缩饼干吃不下了，汤料用完了，爬也差不多爬够了

然而，外面在下雪。。。

都8点了，还是这个德行。。。

10点了。。。

实在等不下去了，用手机跟电脑下了几十盘五子棋了。。。

雪稍微停了，出帐篷勘测一下，外带拉屎。。。

还发现了这个，鸽子？山鸡。。。？

拉完屎决定不等了，收包，收帐篷，下撤

结果收一半又开始下雪。。。。像逃难一样。。。

昨天下午烧了一次垃圾，这一夜产生的垃圾也没法空了，连同那个4L的农夫山泉瓶子、塑料布、1/4罐气，都扔营地了。。。

顶着头皮下降到大斜板，雪停了，出了点太阳，真是心情顿时大好

继续下到4600乱石坡，太阳已经完全出来了

沿着前人轨迹找到了那个石缝

然而旁边就有一个很好下的（黄为刚刚那张的石缝，红色为脚踩的点，沿绿色箭头下脚，一步就到底。。。

然而我还是没下这个，而是边下降边找地方过水走的另一侧

地形后略不表，同来时路

走到传统营地（小卖部）附近开始下雨，一会儿停，一会儿下，一会儿停，一会儿下。。。

于是进了牧民的帐篷帐篷借地下包穿衣服

走到两湖之间又开始出太阳，问了一个小妹妹走湖西侧可以出去不，门口那边封了路呢，小妹妹说可以，并指了我一个踩着石头过河的方向

我顺着方向果然看到很多石头，然后开始过河，结果过到一半开始下雨，踩石头上双手撑着登山杖保持平衡，没法穿雨衣。。。。

简直不能更坑了，前一秒还在出太阳，只穿了长袖T恤，然后过完河就快失温了。。。。

过完河之后一路快走稍微恢复点体温，虽然有信号了也不敢停下来看手机和吃东西，水也喝没了，马不停蹄赶到了景区停车场，又饿又渴又冷

权衡之下先解决口渴问题，找了个小卖部买了瓶水，老板人特好，看我太惨了，让我跟武汉来的一个驴友团一起吃了顿饭，算是老板赏的，几天没吃人饭了，而且今天又特别饿，所以这顿饭吃得特别香、特别撑。。。。

吃完之后，老板（女，一个人住，不方便）把我介绍给隔壁大叔住他们家，结果隔壁大叔帮我背包的时候把手机弄掉了，被一个藏娃捡去，要价1000块，我这手机才买成750块。。。。（二手索尼z3

折腾了好久，终于降价到500元，不折腾了，赎回来吧

跟借宿的大叔谈妥230到阿坝（久治中转），于是睡觉

9月15日 少云，晴

起床吃奶茶泡的黄色淀粉糊状物，外加饼子，又吃撑了。。。

成都-阿坝 黑车130一个人，是早上4点出发 中午12点可到阿坝

阿坝，三百块单独包车，2小时可到年保玉则景区

年保玉则景区回阿坝 50一个人，2小时车程（中午12点之后有车，大概下午2-3点

沟底上升

C1到

BC至C0主要为乱石坡，至瀑布前这一段溪流的左侧好走一点，瀑布下面向右前方向转

到C0途中最危险的是要翻越一个石缝，看起来挺简单，但是这个石缝比较的高，翻上去的平台一侧又是个悬崖，要想背着背包翻上来必须掌握好平衡。

从4000到4600平台这个坡全是塌方的石头，冰瀑那段看似较陡，其实还是有路的。

在快到瀑布的下方的时候，需要向右侧攀升到大岩石上。

由此继续向右侧横移，过一个小溪，翻上一个石堆后，终于到达了平坦的C0营地。

C0营地海拔4500米，位于冰川末端的右侧，有一个足球场大，较平坦，但是地上有不少突出的岩石，能扎帐篷的地方也不是很多，这里有小溪流过，取水方便。

C0再向上，一开始依然是一段很陡的碎石坡路，走起来很费劲。上到碎石坡的顶端后，顺着右边石壁的下方可以上到第二台阶的位置，第二台阶有点像个盆地，乱石纵横，确实如向导说的，不是很好扎营。这里的海拔是4800米。

从第二台阶的左边巨石向上攀登，一直到达冰川的边缘。这里的冰川非常的开阔，下部较平坦，可以称之为第三台阶，上边覆盖了有十几公分的浮雪，这样走起来还不是很滑。不过我们还是换上了冰爪，因为就在不远处的冰川上，我们已经看到了一道道的冰裂缝，我们必须要绕过它们。换好装备，我们7个人做了两个结组，开始前进。

走了有十几米，突然在右边比较平坦的雪地上发现了三顶帐篷，没有人，这应该是上海队的帐篷，他们把这里作为了他们的C1营地。

1.乙酰唑胺（已停产）新药为 醋甲唑胺
加速碳酸盐排泄，提高血液酸性，帮助呼吸，加速适应过程，每天2次，每次125mg，进山前1天开始服用，进山后两天停用，对肺水肿、脑水肿无效
2.硝苯地平

钙离子阻断剂，缓解肺动脉收缩，每12小时20-30mg，缓释为宜，预防和缓解肺水肿，起效较慢，导致低血压，不建议登山使用

西地那非，缓解肺水肿，俗称伟哥，贵。。。

吸入沙美特罗（治疗哮喘药物），预防肺水肿，贵。。。

3.地塞米松（部分人群对此过敏）

类固醇药物降低颅内压，每6小时4mg，能在给药后6小时内起效，缓解脑水肿，赢得下撤时间

4.阿司匹林

进山前1天，每天3次，每次320mg，直到出山，缓解头疼

或者布洛芬（非处方药

5.维生素C

每餐后250mg，维生素E，每餐后100UI，缓解急性高山反应症状

6.水

补充脱水，维持血浆总量，维持携氧能力

7.少吃含有亚硝酸盐和亚硫酸盐的食物

避免偏头疼，例如 奶酪、巧克力、柑橘、卤肉、味精.

8.白天食用易消化、高GI值食物（单糖、碳水化合物）

少食多餐，减少肠胃负担供血需求

9.晚上食用蛋白质、脂肪含量高的食物

以持续供能度过漫漫长夜

如果食物单一注意补充维生素、电解质（每升水添加1.5g低钠盐，可同时补充Mg、K）
