Nintendo-Switch-4G热点联机对战的一种解决方案

Posted on 2018-01-20 | In 3C & 数码 | | Visitors:
Words count in article: 1.2k

部分地区和运营商的手机卡（上网卡）在开手机热点或是插入随身WiFi（mifi、4G移动路由器）的时候，因为拿不到公网IP，NAT层数较多，无法联机（比如喷射战士2.。。。直连电信NAT Type D）

除了几乎不可能的给运营商打电话投诉这一条途径，还有就是在路由器上挂SS（需要带UDP转发的SS服务器）并启动路由器上的UDP全局转发（游戏模式）这一功能（梅林、Padavan均支持）

理论上只要肯折腾，openwrt的路由器也支持，可以直接买这个，不过因为政策原因厂家也不敢把SS功能做得太好（https://www.gl-inet.com/forums/topic/shadowsocks支持/），所以需要自己弄

然而，首先我穷，600太贵了，其次懒得再去学习openwrt了，于是选择了熟悉的Padavan

首先需要一个乐携Nexx WT3020F，淘宝售价88元，mt7620处理器16M/64M，1WAN 1LAN 1USB，5V MicroUSB供电，体积比火柴盒稍大

然后按此方法刷Padavan，全文摘抄如下：

【刷入Breed】

1. 将U盘格式化为FAT32

2. 将breed.bin 和 WT3020H.trx 拷贝至U盘根目录，并连接至路由器USB接口

3. 路由器通电，稳定后，通过命令提示符Telnet 192.168.8.1 （非摘抄的注释：Win10 2017秋季更新之后，开启telnet需要从控制面板的程序和功能进入，从文件浏览器的卸载程序进不去了）

Login: nexxadmin

Password: y1n2inc.com0755

4. 进入U盘目录 cd /media/sda/sda1（非摘抄的注释：如果多次插拔U盘，会变成sda2 sda3 sda4.。。。最好一次搞定或者重启路由器

5. 可以考虑备份原厂固件，操作命令逻辑大致如下（非必须步骤）

for i in 0 1 2 3; do cat /dev/mtdblock\$i > mtdblock\$i.bin; done

6. 刷入Breed

mtd_write -r -e mtd0 write breed.bin mtd0（非摘抄的注释：一定要找对breed版本，不然会变砖

稍等几秒，即可完成Breed刷入

【刷入固件】

1. 保持U盘连接至路由器USB接口，通过命令提示符Telnet 192.168.8.1

Login: nexxadmin

Password: y1n2inc.com0755

2. 进入U盘目录

cd /media/sda/sda1

3. 刷入固件

mtd_write -r -e mtd3 write WT3020H.trx mtd3

大约1分钟左右，固件刷新完成

路由器重启后，登陆 192.168.123.1 即可登陆新系统后台管理.

【备注】

Breed.bin 版本：breed-mt7620-reset1.bin （Breed r936 [2016-08-25]）

参考：http://www.right.com.cn/forum/thread-161906-1-1.html

（非摘抄的注释：快速下载链接 https://breed.hackpascal.net/breed-mt7620-reset1.bin

WT3020H.trx 版本：RT-N14U-WT3020H-64M_3.4.3.9-099.trx （Padavan_20160826）

参考：http://www.right.com.cn/forum/thread-161324-1-1.html

（非摘抄的注释：快速下载链接 http://opt.cn2qq.com/padavan/RT-N14U-WT3020H-64M_3.4.3.9-099.trx

摘抄完

刷完之后，你还需要一个USB的4G网卡，淘宝关键字为 4G 卡托 ，买非WiFI版（电脑版）就行，联通3G 4G的便宜，四五十都有，电信4G的贵些100+了

不过，之前试了下紫米那个4G无线路由器充当USB网卡，结果几分钟断线一次。。。后来换了华为的一个4G无线路由器来充当USB网卡，就正常了可以持续连接不掉线

最后在咸鱼上170买了个华为E8372h-608水货（带WiFi版），稳定点，不想试运气了。。。

USB 4G网卡插入后路由器之后，在路由器设置中的 高级设置 USB应用程序 选择启动 USB调制解调器

国家、运营商选你对应的，接入点联通wonet（据说比3gnet快些），移动cmnet，电信ctlte（据说比ctnet快些）

然后回到 网络地图，查看点击中间那一排图标最底下一个 USB调制解调器状态，正常的话就说明网卡连上了，没连上的话去翻日志看看有啥问题虽然看了也不会懂的（我）。。。

再点击中间那一排图标最上面一个 外部网络状态，如果看到获得到了IP，就说明连上4G网络了

最后配置 扩展功能 5had0w50cks ，记得开高级功能里的游戏模式（UDP转发），端口留空全部转发，就可以了

紫米4G路由器给3020供电，同时充当USB有线网卡，然而会断线。。。

后改为华为的设备就好了，但是需要再加一个移动电源

PS：路由器和USB网卡之间那根线有点挑线，如果插上路由器找不到网卡的话，可以换个线试试

PS2：据说可以在手机上开热点，使用带udp转发的ss、ssr，然后用一个叫 VPN tether 的软件进行中继，不过我的手机上没有试验成功
