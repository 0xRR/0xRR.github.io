2023克里雅古道穿越计划

发表于 2023-02-23 | 分类于 摩托 | | 阅读次数:
字数统计: 2.9k

这条路眼馋好多年了，迫于瘟疫带来的封锁，一直未能出行，在2023年终于可以得以落实计划，但愿能够成行。

自己的计划，酌情参考，不对文中信息准确性负责。

  • 自然环境

经过在网上搜索文字资料，以及观看了B站、YTB上的视频之后，大概有这么几个队伍走过： 1.王铁男，4、5月走的，西藏进，地上有雪，红土达板第一次没上得去 2.成都川A大军，应该是更早一点4月初的样子吧，不太确定，新疆进，库拉普河（普鲁河）里冰大，过河凿冰凿了一天 3.长城赞助的车队？开的哈佛H5，哪里路烂往哪里开，过河陷车，泥潭陷车，而且全是没车辙的地方，可能是性能测试？ 4.还有一队人，雨季，新疆进，库拉普河（普鲁河）里水大 6.另外一队人，10月下旬，西藏进，遇到暴雪，先遣乡附近路上有大量积雪，勉强通行，红土达板没上去，连接线去G219又遇到塌方 5.其他还有几个队伍，比较顺利，没有参考价值

结论，不要4、5月去，雪还没化，库拉普河（普鲁河）里冰大；也不要雨季678去，水大过河困难（根据OpenStreetMap提供的数据，在库拉普河（普鲁河）需要过河10次左右）；最好就是9月下旬到10月初这一段时间；再后面就有可能遇到下雪，天气也冷了。

还有一个队伍遇到了熊和狼，不过那是在远离克里雅古道的地方，在克里雅古道主路上只有少数队伍报告远处有狼。

  • 路况

G219泉水湖加油站到兽形湖120公里为铺装路（大概），或改则县经先遣乡到兽形湖530公里也是铺装路（G216）。 兽形湖往北开始的370公里到于田，其中到普鲁村260公里（中间红色轨迹）为非铺装路（兽形湖往北可能有少量铺装路，大概）。

从西藏这个方向，到红土达板之前有个陡上坡，2.5公里爬升400米，平均坡度16%，坡角是9度，似乎也不是很陡（对摩托车而言）……顺着车辙走还算比较平整，不过可能有浮土或是泥浆（看天气情况）。

然后一路无难点，直到快要出山时的库拉普河（普鲁河）过河，如果水大就比较难走。

民丰出来有个库亚克大峡谷线，黄色部分是G216，应该是铺装路，紫色的连接线为非铺装，可作为红土达板翻越失败的后撤路线。月牙湖到路口（橙色200公里）也是G216大概率已铺装。

  • 检查站

1.先遣乡出来100多公里，阿鲁错东南方向有一座山，山脚下有个检查站，但是因为这条路是在G216上，问到就说走G216去民丰的，应该不太会强力阻止，实在不行也可以往左往阿鲁错方向沿着湖绕行。

2.兽形湖有个检查站，有可能出于安全原因阻止我单车进入克里雅古道，考虑从左侧绕过，或者如果底面条件允许，可以提前从邦达错左岸或右岸直接插过去，就不用到兽形湖了。

3.阿拉叫依检查站，肯定不让我单车进克里雅，这就是我从西藏方向进的原因了，出不可能不让我出了，毕竟现在没有防控引起的封锁了。

  • 风险评估
  1. 油料不足
  2. 普鲁河过河
  3. 车辆故障
  4. 野生动物

对应方案：

  1. 如果从G219泉水湖加油站加油后，到于田一共490公里，其中非铺装260，铺装130，按300排量单缸水冷电喷计算，非铺装油耗4.5，铺装油耗3.5，这样需要2.6 x 4.5 + 1.3 x 3.5 = 16.25 L。 但是我是从改则县开始走，这样一共900公里，其中非铺装260，不过按悲观情况300公里非铺装，600公里铺装，这样需要 3 x 4.5 + 6 x 3.5 = 34.5 L。 我的车主油箱 11 L 按 10 L 计算，副油箱按 7 L 计算，这样还需要外带 18 L 汽油才够。 假设先遣乡和普鲁村能够买到散装油，这样减去240公里铺装路，仍然需要外带10L油。 所以得按最坏情况考虑，需要在左右边包各放 10 L 油，考虑两个农夫山区 5L 桶，或者是一个 10 L 的油桶，有待前期实际测试。

  2. 穿捞鱼那种连体长裤，水深的话先摸一遍，实在不行等晚上半夜水小的时候走

  3. 出发前全车损耗件全部换一遍，维修工具和维修备件带齐；带太阳能电池板和登山包，发动机要是坏了只有自己走出来，同时现在也开始体能训练，做到高海拔每天20公里平路徒步的水平。最霉的话在最中间烂，走出来130公里到兽型湖或者啊辣椒衣检查站，应该可以讨口饭吃了。
  4. 带火炮和汽笛，先远远地吓唬吓唬，他们多半也不会主动搞我。
  • 行程安排

尽量在先遣乡过夜，寻找汽油（多半是散装）补给，然后第二天一早出发，400公里铺装路到兽型湖检查站过夜（或在接近兽型湖附近寻找废弃人工建筑）。

兽型湖一大早起来（趁着工作人员没起床？）就开始260公里克里雅古道核心路段的穿越，争取一天跑到阿拉叫衣（普鲁村）。如果此路段中途耽误，则寻找人工建筑残骸扎营。

在普鲁视油料剩余情况是否加油，然后走到G315右拐到民丰县补给完整，并过夜。

次日沿G216过道返回西藏，并在三叉路口离开G216，走连接线返回克里雅古道红土达板，再沿着来时的路返回兽型湖检查站。如果此路段中途耽误，则寻找人工建筑残骸扎营（不过此线路上估计没有人工建筑残骸

次日再由兽型湖检查站往泉水湖方向，走上G219过道。

PS：此段废弃人工建筑具体位置需进一步落实。

  • 车型选择及改装

从稳定性考虑尽量选择日系进口，从重量和功率的平衡性考虑应选择300/450排量，太重倒车不好扶车，排量太小非铺装爬坡脱困都困难。 250风冷电喷应该是排量的下限了，像crf190L 豪爵nk150最好不要去 ( 欢迎nk150车手实操打脸。。。

轮胎一定要选硬派拉力胎，至少是50/50的，参考型号普利司通 AX41，米其林anakee wild，象牌karoo4，Mitas E09 E10、E07勉强，后轮神冈E805凑合泥地差点。不要选那些号称adv拉力胎其实更偏向公路的轮胎，那些是宝马水鸟那种车用的。 三种adv轮胎的介绍：https://www.youtube.com/watch?v=68JTYYCJSVc 各种轮胎测评：https://www.chapmoto.com/blog/motorcycle-gear-guides/adv-motorcycle-tires/

我前轮是米其林anakee wild（考虑换普利司通 AX41试试），后轮是国产高齿漏气保用真空胎，加自补液。

如果对车圈撞变形后真空胎不可用的问题不放心，可以选择非真空胎，但是我认为非真空胎被扎的概率远大于车圈撞变形的概率。跑摩旅穿越又不是达喀尔，有大石头时候慢点跑就是了，而地上的钉子可是没办法躲的。

前轮无脑90/90-21，后轮17 18都行，不差那1寸高度，主要是看胎齿形状，和宽度，宽度建议根据车辆重量来。满油含人含行李总量300kg以内选择130宽度，以上选择140宽度。 因为轮胎在非铺装上抓地力主要是轮胎对地面的压强（车重除以轮胎宽度）决定，胎窄过泥地刨泥巴得劲儿，胎宽过砂石河滩不容易陷车，而西藏阿里地区的线路的砂石河滩多于烂泥地，所以应当选择稍微宽一点的轮胎。

其他改装尽量不要增加重量，用边包而不是边箱，尾箱尽量轻小只用来放一些常取用的东西。重东西尽量往边包放，这样重心低好操控，而且翻车也好扶。

副油箱（外带油桶）一定得安排上，考虑到加油站不会给外带油桶加油，建议在主油箱上开孔加装一个放油口，可以省去频繁抽油的麻烦。

  • 其他需要注意的问题

对于路况难度描述上，可能存在偏差，建议把b站和ytb所有视频都看一遍再自己考量。如果平时有玩越野摩托车的话，那应该是比较轻松了，如果平时是骑公路adv车型很少走烂路的话，三思。

横向对比，在没有下雨没有下雪的9-10月而言，比川藏中线、阿里中线略难，远难于丙察察丙察左宝康线宝金线，与敦煌-野马河-哈拉湖西线穿越持平；但是季节不对，天气不好，难度上限远高于上述线路。
